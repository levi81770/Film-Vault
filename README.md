# 🎬 Film Vault

![Film Vault Screenshot](./assets/screenshot.png)
## Description

Film Vault is a full-stack web application where movie lovers can discover, share, and review films. 

Users can browse a community-built database of movies and actors, add their own entries, and leave reviews on films they've watched.
